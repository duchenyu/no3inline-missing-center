# 数据充分性审计：当前数据集能否支撑这些渐近假说的建立？

> ⚠️ **纠错（2026-07-09）**：本审计将"D₆ 主导（稀疏性）"评为"中（有界区间）"——
> 实为**假（代码 bug）**。原"≥89% D₆ 主导"来自 `collinear3` 只查 6 点中前 3 点的残破函数
> （见 `analysis/d6_dominance_correction.md`）。正确测量：D₆ 占比随 n 单调降至 ~28%，
> 高斜率共线主导（~72%）。故审计结论中"稀疏约束→0 成立且可作一般渐近论证"一项**作废**；
> 真实约束分散于高斜率方向且随 n 增长，反而支持阈值/相变叙事。

> 同行评审视角（Math skill：区分 定理 / 猜想 / 开放问题；承认不确定性；不模糊已证与未证）。
> 问题：凭 n≤76（多数对称类远少于此）的有限数据，能否**严谨地建立**
> “阈值相变 / 模障碍同余类 / 稀疏约束→0 / 正维族” 等渐近假说？
>
> **一句话结论：数据可以*生成*这些假说，但无法*确立*它们。若干假说目前立在极薄的冰上。**

## 3. 我们正在犯的三个致命统计错误

### 陷阱 1：跨 16 格真空外推
C₄ 全枚举止于 **n=56（10,441 解）**，下一个数据点直接跳到 **n=72（1 解）**。
中间 n=58..70 **零全枚举**。所谓“相变在 n≈56–72”的推断，跨在一道 16 格的空洞之上，
且仅由 **3 个稀疏/搜索不完整点**支撑。
> 定位阈值需要阈值**两侧**都有数据。我们只有“上升段 26 点”+“上方 3 个模糊点”。
> **统计功效 ≈ 0**：无法定位、甚至无法确认阈值存在。

### 陷阱 2：事后拟合（hypothesis generated by data 不能由同一数据验证）
我们先看到 n=56 峰值、n=72 跌落，**然后**提出阈值假说。假说由数据生成，
数据便不能再充当它的验证。正确做法：用 n≤48 拟合模型 → 预测 n=50..56 → 检验。
但我们的指数模型是在**上升段**（计数仍在涨）拟合的，根本没触及“ relevant 区域”，
等于拿 logistic 曲线前半段去断言后半段会塌——典型误用。

### 陷阱 3：把“搜索不完整”读作“UNSAT”
n=74“0 解”、n=76“2h 无”**不是否定结果**，而是**不完整搜索**。
Heule 在 n=72 找到 1 解近乎奇迹式抽样，不是典型样本。
把“没找到”当“不存在”，是 II 类错误风险。rot2 的 n=31 UNSAT 是**穷尽**的（可信），
但**仅此 1 点**——单点无法支撑“同余类障碍”。

### 附：对称类→一般的范畴错误
rot2 UNSAT、C₄ 塌缩都是**子类**现象。即使 rot2 对无穷多 n UNSAT，
也与一般 D(n) 无关（iden 类才是 D(n) 的度量）。这正是我们已在 v₂ 事件上栽过的
“相关性 ≠ 因果性”的翻版，必须同样警惕。

## 4. 让每个假说“经得起”的最小实验程序（功效分析）

| 假说 | 为确立所需的最小数据 / 证明 |
|------|----------------------------|
| **B 阈值** | 关闭 n=58..72 的 16 格真空（全枚举或严格穷尽）；在阈值**上方**取 ≥10 个单调下降的成功概率点；再用 n≤48 拟合外推 n=50..56 做**留出验证** |
| **A 模障碍** | rot2 的 SAT/UNSAT 至少覆盖 n=31,33,35,…, 至可枚举上限；需 ≥5–10 个 UNSAT 点且呈现周期/同余规律；或给出**解析证明**（目前机制未知） |
| **稀疏→0** | **解析证明** D₆ 主导与低斜率空性对**所有** n 成立（把经验规律转为定理，彻底摆脱数据依赖） |
| **D 正维族** | 不止 1 个样本：在 n=72 之外取多个大 n 解（或有理论证明切空间正维并延拓到整数格） |
| **C 吸收** | 在 n=58..72 实测冲突密度以收窄指数 CI（当前 −1.5 的 95% CI 约 [−1.1,−1.9]）；“冲突增长”稳，“速率”不确 |

**定量说明**：拟合幂律指数 α 要达到 ±0.3 精度，需 ~1.5–2 个数量级的 n 范围且密采样。
我们 n 仅跨 12→72（<1 个数量级），高 n 稀疏。故“期望冲突∝N^1.5 增长”这一**趋势**稳健，
但**速率**与“吸收是否成功”远未确定。

## 5. 正确定位：数据是“假说发电机 + 证明靶标”，不是“验证器”

诚实用法：
> “n≤56 数据暗示 n≈56–72 附近可能存在阈值；我们提出猜想 X；**检验方法**是计算 Y（关闭真空 / 留出验证）。”

**不可接受**的用法：
> “因为 n=72 仅 1 解，所以阈值真实存在、n*≈56–72。”（用生成假说的同一数据“验证”它）

**对本项目先前备忘录（research_directions_2026-07-09.md）的修正**：
- “杠杆判定”中把 n=31 与 n≈56–72 称为“两个精确提出的假说源”**措辞过强**；
  更准确应为“两个**需大幅加强数据或证明**才能成立的经验迹象”。
- 方向 A、B 的置信度应明确标注为**弱/极弱**，而非可作为“撬棍落点”。
- 真正**数据稳健**的资产只有：D₆ 主导（n≤32）、低斜率空性（n≤60）、
  rot2 n=31 单点 UNSAT、C₄ n≤56 丰富 + n=72 稀有。

## 6. 建议的下一步（按数据伦理排序）

1. **先关真空**：把 C₄ 全枚举推到 n=58..72（或严格证明 UNSAT），消除 16 格空洞。
2. **对称类留出不交叉验证**：n≤48 拟合并外推，检验预言能力，再谈阈值。
3. **模障碍扩点**：rot2 扫 n=31..上限，积累 UNSAT 点而非依赖单点。
4. **把稳健经验规律转定理**：证明 D₆ 主导 / 低斜率空性对所有 n（这是唯一能摆脱数据依赖的杠杆）。
5. **写作纪律**：所有假说标注【猜想】+ 样本量 + 支撑强度（弱/中/强），绝不与【已证】混淆。

*审计结论：当前数据集是优秀的**假说来源**与**证明靶标**，但**不足以确立**任何所列渐近假说。
在关闭真空、积累 UNSAT 多点、或给出解析证明之前，阈值与模障碍只能作为待检验猜想存在。*

## 7. 追加：低斜率空性 — 实测修正（审计催生的一次成功升级）

对“低斜率空性”项做严格核查（见 `low_slope_parity_theorem.md`），结果**推翻了原声称的不完整表述**，但将其升级为可证定理：

- 原声称“Hₙ[L] 在 n≤60 内为空”**不准确**：实测 n=6..60 逐 n 穷举发现，
  **偶数 n 全空，奇数 n≥5 全有反例**。原 TARGETS 只取偶数（12,16,…,60），掩盖了奇偶依赖。
- **解析证明（奇偶律）**：
  - 偶数 n=2m：任意格点中心方向分子两坐标皆奇 → 归约后仍两坐标奇；而 LOW 每个方向至少一坐标偶
    → 偶数 n 根本不存在低斜率轨道点 → Hₙ[L] 平凡空（对所有偶数 n 成立）。
  - 奇数 n=2m+1 (m≥2)：竖直线三点 (m+2,m−1),(m+2,m),(m+2,m+1) 构成显式超边 → 对所有奇数 n≥5 非空。
- `count_low_pts` 实测：偶数 n→0；奇数 n=5,7,11,21,61→16,28,44,104,320，与证明一致。

**方法论寓意**：这条子问题恰好演示了审计的两条相反教训——
(1) **反面**：仅测偶数 n 的数据不能支撑“对一般 n 成立”的泛化（原声称之误）；
(2) **正面**：完整核查（含奇数）+ 解析论证后，经验声称升级为带奇偶判定的精确定理，
    且**不再依赖数据**。这正是审计建议“把稳健经验规律转为定理”的实例。
- **对“稀疏约束”叙事的影响**：低斜率共线在偶数 n“缺席”是**偶数专属奇偶巧合**，
  非一般渐近稀疏；奇数 n 从 n=5 起即丰富存在。故“危险集中于 D₆/低斜率空”不能作一般渐近论证，仅偶数 n 局部成立。
- 该子问题现**已完全解决**（带奇偶判定），移出开放方向清单。
