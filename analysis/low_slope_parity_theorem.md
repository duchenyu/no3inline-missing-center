# 低斜率空性：奇偶定理（计算机辅助 + 解析证明）

> 修正 `hypergraph_theory.md` 经验事实 2（“Hₙ[L] 在 n≤60 内为空”）。
> 该声称原为**逐 n 穷举验证**（对测试的每个 n 是严格的），但 TARGETS 只取了
> **偶数** n=12,16,…,60，遗漏了奇数，导致陈述不完整。
> 本文件给出精确结论，并解析证明之。

## 1. 定义（取自 hypergraph_low_slope.py，即 Hₙ^dir 模型 Model 1）

- 网格点 P=(r,c) 的**中心方向**：`dir(P) = reduced(2r-(n-1), 2c-(n-1))`（gcd 归约，规范化半平面）。
- 低斜率方向集 `LOW = {(1,0),(0,1),(2,1),(1,2),(2,-1),(1,-2),(3,2),(2,3),(3,-2),(2,-3)}`。
- 低斜率 C₂-轨道点：中心方向 ∈ LOW 的网格点。
- 超边：三个低斜率轨道点 P₁,P₂,P₃ 共线（离心），且连线方向 `ld=reduced(P₂−P₁)` 也 ∈ LOW。
- “Hₙ[L] 空” = 不存在这样的超边。

## 2. 定理（精确陈述）

**定理（低斜率空性 — 奇偶律）。**
令 n 为偶数。则 n×n 网格中**没有任何**网格点的中心方向属于 LOW。
因此 Hₙ^dir[L] 对**所有偶数 n 恒为空**（平凡空）。
反之，对每个奇数 n ≥ 5，Hₙ^dir[L] **非空**；显式见证为竖直线三点
> P = (m+2, m−1), (m+2, m), (m+2, m+1)，其中 n = 2m+1。

其三个中心方向分别为 `(2,1)`, `(1,0)`, `(2,−1)`（均 ∈ LOW），连线方向 `(0,1)`（∈ LOW）。

**证明（偶数 n — 奇偶性）。**
设 n=2m 为偶数。任意网格点 (r,c) 的中心方向分子为
`(2r−(2m−1), 2c−(2m−1))`，两坐标均为**奇数**。
归约时除以 gcd；由于两坐标皆奇，gcd 为奇数，故归约后的方向 (a,b) 仍满足
**a,b 均为奇数**。
但 LOW 中每个方向至少有一个偶坐标（`(1,0),(0,1)` 有零；`(2,±1),(1,±2),(3,±2),(2,±3)` 均一奇一偶）。
故归约方向不可能落在 LOW 中。于是偶数 n 时低斜率轨道点集为空，Hₙ[L] 平凡为空。∎

**证明（奇数 n≥5 — 显式构造）。**
设 n=2m+1（m≥2）。取三点 P₁=(m+2,m−1), P₂=(m+2,m), P₃=(m+2,m+1)。
中心为 (m,m)，故
- P₁ 的偏移 (2,−1) → dir=(2,−1) ∈ LOW；
- P₂ 的偏移 (2, 0) → dir=(1,0) ∈ LOW；
- P₃ 的偏移 (2, 1) → dir=(2,1) ∈ LOW；
- 三点共线于竖直线 x=m+2，连线方向 ld=(0,1) ∈ LOW。
当 m≥2（即 n≥5）时三点均在网格内且互异，且不在过心线上。故构成超边。∎

## 3. 计算验证（穷举，非采样）

`low_slope_threshold.py` 对 n=5..60 逐 n 穷举（覆盖所有网格点/轨道）：

| 性 | n 范围 | 结果 |
|----|--------|------|
| 偶 | 6,8,10,…,60 | **全部 EMPTY**（低斜率轨道点数=0，已另算验证） |
| 奇 | 5,7,9,…,59 | **全部 FOUND**（首反例即上述竖直线构造） |

`count_low_pts(n)` 实测：n=6,8,12,20,60 → 0；n=5,7,11,21,61 → 16,28,44,104,320。
与奇偶性证明完全一致。

## 4. 对原研究叙事的修正

1. **原声称“Hₙ[L] 在 n≤60 内为空”不准确**：它对所有**偶数** n 成立（且为结构性的平凡空），
   但对所有**奇数 n≥5 失败**。原 TARGETS 仅取偶数，掩盖了奇偶依赖。
2. **这恰好印证了数据充分性审计的核心警告**：仅测偶数 n 的数据，
   不能支撑“对一般 n 成立”的泛化声称。完整核查（含奇数）后，
   经验声称升级为带奇偶判定的精确定理——而非被推翻。
3. **对“稀疏约束”叙事的影响**：低斜率共线在偶数 n “缺席”是**偶数专属的奇偶巧合**，
   不是一般的渐近稀疏性；奇数 n 从 n=5 起低斜率共线即丰富存在。
   因此“危险集中于 D₆ / 低斜率空”不能用作一般渐近稀疏论证，仅对偶数 n 局部成立。
4. **积极面**：低斜率空性（偶数 n）现为**可证定理**，不再依赖数据；
   这正是审计建议“把稳健经验规律转为定理”的实例——且揭示原陈述需补奇偶条件。

## 5. 遗留开放问题

- 偶数 n 的空性是结构性（奇偶）且对所有偶数成立，**无需更多数据**。
- 奇数 n 的非空性已由显式构造对所有 n≥5 证出，**亦无需更多数据**。
- 因此“低斜率空性”这一子问题已**完全解决**（带奇偶判定），不再属于开放方向。
  真正仍开放的是 **C₄ 阈值 / 模障碍**（见 data_adequacy_audit）。

  > ⚠️ 注（2026-07-09）：原"D₆ 主导的密度是否对所有 n 保持 ≥89%"已**撤回**——

  > `analysis/d6_dominance_correction.md`）。正确结论：D₆ 占比随 n 单调降至 ~28%，
  > 高斜率共线主导。故不再作为开放方向。
