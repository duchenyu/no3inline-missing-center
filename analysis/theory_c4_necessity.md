# ★★★ C4 恒等式强制性定理（实证定理）
## No-Three-In-Line 结构理论核心发现：大 n 时 C4 恒等式为强制性

> **声明：** 这是一个基于全量已知数据的**实证定理**——对所有已知 NTIL 解成立。解析证明仍 OPEN。

---

### 定理陈述

对 $n \ge 33$ 的全部已知 NTIL 解 $S$（$|S|=2n$），
$S$ 对应的 $(π,σ)$ 函数满足 **C4 恒等式**：

$$\pi(n-1-i) + \sigma(i) = n-1, \quad \forall i \in [0,n-1]$$

等价地：
$$\{S \text{ 是 } n\ge 33 \text{ 的任意 } 2n \text{ 解}\} \iff \{S \text{ 满足 C4 恒等式}\}$$

|-------|------|----------------|------|
| 7-15 | ~10,000 | 5-45%（混合） | 大量非 C4 解存在 |
| 16-20 | ~25k+ | ~0%（非 C4 主导） | 几乎所有解非 C4 |
| 21-24 | ~11k | ~99%（C4 主导） | 少数非 C4（~0.5-1.5%） |
| **25-30** | **~25k+** | **100%** | **0** |
| 31-32 | 247 | 10-58%（混合） | dia1 类的非 C4 解 |
| **33+** | **~3,000+** | **100%** | **0** |

### 相变机制解释

**过渡的驱动因素**是非 C4 对称类的逐步灭绝：

| 对称类 | 最后出现 n | 满足 C4 恒等式？ |
|--------|----------|----------------|
| iden（无对称） | 27 | ✗ |
| ort1（正交） | 28 | ✗ |
| rot2（180°旋转） | 33 | **✓**（存活至 n≥33 的强对称类全部满足） |
| dia1（单对角） | 32 | ✗ |
| dia2（双对角） | 44+ | **✓** |
| rct4（奇数 C4） | 53+ | **✓** |
| rot4（偶数 C4） | 72+ | **✓** |

**核心观察**：在 n≥33 时，**所有存活的对称类**（rot4、rct4、dia2、rot2）都满足 C4 恒等式。而不满足的类（iden、ort1、dia1）已经在 n≤32 灭绝。

---

### 定理的推论

1. **D(n)=2n 问题的大 n 归约**：对所有 n≥33，D(n)=2n 的存在性等价于**C4 恒等式解的存在性**。

2. **签名空间简化**：n≥33 时，签名必须同时满足：
   - Motzkin 路径条件（Th-17）
   - C4 互补性（$c\in L \iff n-1-c\in R$）
   
3. **k-值收敛**：由 Th-17 推论，$k = |L| \sim n/\pi$，配合 C4 互补性，完整签名由前 $\lceil n/2\rceil$ 行的签名唯一确定。

### 开放问题

1. **为什么 n=31,32 有非 C4 解但 n=33 没有？**——这些解来自 dia1 类（最后出现 n=32），但为何 dia1 不能在 n=33 存在？

2. **是否存在 n≥33 的非 C4 解未被发现？**——当前搜索不保证完备，但 Flammenkamp 的多策略搜索对 n≤44 应较彻底。

3. **C4 强制性的解析证明**——需要一个基于组合参数的证明：证明非 C4 解的 $k=|L|$ 有全局上界（审计修正：观测上限 14，n=28 发现 k=14 的非 C4 解），而 Motzkin 路径的 $k$ 随 $n$ 增长，因此大 $n$ 时非 C4 解不可能。
