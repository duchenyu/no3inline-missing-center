## rct4 的未来搜索边界

rct4 在 n=53 之后（n=55,57,59...）的情况完全未知——既无法验存在也不能证不存在。这是当前最有计算价值的研究方向之一。

**关键未知**：
- rct4 n=55: 未搜索
- rct4 n=57: 未搜索
- rct4 n=59: 未搜索
- rct4 n=61+: 搜索成本极高（mvr 用 CUDA 搜到 n=53）

## 数据文件索引

### Flammenkamp 原生文件（编码格式, 80-char alphabet）
- `n{n}_{class}` — 完整枚举
- `n{n}_{class}.few` — 部分抽样
- `n{n}_full` — 全 D4 对称

### mvr 补充文件（坐标格式, space-separated）
- `n{n}_rct4.mvr` — c4near（rct4 的扩展）
- `n{n}_rot4.mvr` — c4（rot4 的扩展）
- `n{n}_sporadic.mvr` — 独立解

### 加载方式
```python
from analysis.unified_loader import load_n, list_available
sols, sources = load_n(47)  # 加载 n=47 全部解
avail = list_available()    # 列出所有可用数据
```
