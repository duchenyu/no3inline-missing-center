# rot4 方向结构定理（Th-50 / Th-51）

本文记录两条关于 rot4 解方向（direction）结构的定理，均建立于已证无损双射
Th-44：`rot4 解 ⇔ 2-正则伪图 G on m=n/2 顶点 + 每格奇标度 g`，其中格 `(x,y)`
的中心相对方向为 `(a,b) = (dx/g, dy/g)`，奇本原，`dx = 2x-(n-1), dy = 2y-(n-1)`，
`g = gcd(dx,dy)`。每个格提升为以 `C=((n-1)/2,(n-1)/2)` 为中心的 4 点正方形
（C₄ 轨道）：`(a,b), (-b,a), (-a,-b), (b,-a)`（已按 `g/2` 缩放，方向仅由 `(a,b)` 决定）。

---

## Th-50 — 方向轨道互斥（必要条件）

**命题.** 在任一合法 rot4 解中，m 个格的方向轨道
`O(c) = {(a,b), (-b,a), (-a,-b), (b,-a)}`（c 为格方向）两两不交。
换言之，任意两不同格的方向轨道不能共享任何方向（模 C₄ 旋转与符号）。

**证明（过心线论证）.** 取任意方向 `(a,b)`。过中心 `C` 且方向为 `(a,b)` 的直线
`L` 上，格 `c` 的 4 个提升点中，恰有 2 个落在 `L` 上：即方向为 `(a,b)` 与
`(-a,-b)` 的那一对对跖点（其余两点方向 `(-b,a),(b,-a)` 垂直于 `L`，不在 `L` 上）。
故 `L` 已含该格的 2 个共线点。

若存在第二格 `c'` 与 `c` 共享方向轨道，则 `c'` 也含方向 `(a,b)`（或 `(-a,-b)`）
的 2 个提升点，且它们同样落于 `L`。于是 `L` 上共有 4 个解点 → 至少有 3 个共线
→ 违反 NTIL。故不同格的方向轨道必须两两不交。∎

**验证.** 对全部 **21,698** 个缓存 rot4 解（n=8..72）解码并校验方向轨道互斥性：
**0 失配**。

**非充分性（gap 分析）.** 在小 m 上枚举所有 2-正则伪图并检验：满足方向互斥者
绝大多数仍含共线三点。例如
- m=4 (n=8)：互斥伪图 83，其中 75 仍共线；
- m=5：互斥 667，共线 654；
- m=6：互斥 7643，共线 7635；
- m=7：互斥 101749，共线 **101723**。

故 Th-50 是**必要非充分**条件——它仅排除约 0.03% 的伪图，真正约束来自非过心线
的共振（即 Th-42 危险超图）。

## Th-51 — 斜率精确禁令（修正 Th-24）

记忆中 Th-24「低斜率空集：|a|≤3,|b|≤3 除 D₆ 为空」**已被证伪**。精确禁令如下。

记格方向 `(a,b)`（奇本原）。

**(i) 斜率 0 与 ∞ 被奇偶律禁止.** 由 Th-23 所有 `(a,b)` 奇，故 `a≠0, b≠0`，
恒无水平/竖直方向。

**(ii) 斜率 −1（a = −b）被坐标论证禁止.** 若 `a = -b` 则 `dx = -dy`，即
`2x-(n-1) = -(2y-(n-1)) ⇒ x+y = n-1 = 2m-1`。
但 `x,y ∈ [0, m-1] ⇒ x+y ≤ 2m-2 < 2m-1`，矛盾。故**不存在**斜率 −1 的格。
全缓存验证：**0** 个 `a=-b` 格。

**(iii) 斜率 1（a = b）⇔ 环格 ⇔ 对角解.** 若 `a=b` 且本原奇，则 `a=b=±1`，
从而 `dx = ±g = dy ⇒ x = y`，即该格为环格 `(x,x)`。由 Th-48，环格存在
⇔ 解占对角。故**斜率 1 仅出现于对角（含环）解**。全缓存验证：**0** 个
非环格的 `a=b` 格。

**(iv) 反例：Th-24 的「低斜率空」错误.** 方向 `(−1,−3)` 与 `(−3,−1)`（均满足
`max(|a|,|b|)≤3` 且非 `(±1,±1)`）在缓存中出现 **4735 + 2785 = 7520** 次。
故原声称的区域为空**不成立**。

**结论（精确禁止斜率集）.**
> rot4 解的方向斜率中，真正被全局禁止的只有 **{0, ∞, −1}**（前二者由奇偶律，
> 后者由坐标论证）；斜率 **1 仅当对角（环）解**出现；其余所有奇本原斜率
> （包括 `(3,±1)`, `(1,±3)`, `(5,±3)` 等）均允许且常见。

## 附：两条实证定律（尚未证明，供后续攻击）

1. **对角占比 ≈ 1/2.** 全部 21,698 解中 11,176 (51.5%) 含环（对角）。在采样充分的大
   n（n=54: 52.1%，n=56: 51.4%）稳定在 ~51–52%，强烈暗示渐近 1/2。结构解释：环占用其顶点
   的全部 2 度且不引入额外禁止（对角方向轨道不被其他格使用，因斜率 −1 永不发生），故
   「放一个环」近似自由——这解释了为何约半数解对角。

2. **最大方向幅度 = n−1.** 每个被取样的 n（8..72）均有 `max(|a|,|b|) = n-1`。
   即解总用到近乎极端的方向。推论：方向轨道的供给（随幅度二次增长）始终充足，
   **不存在「方向耗尽」导致大 n 不可行的证明路线**——C₄ 必要性或 Guy–Kelly 的失败
   必源于非过心线的共振结构（Th-42 / Th-50 的非充分部分），而非方向短缺。
