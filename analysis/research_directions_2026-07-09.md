## 0. 术语先对齐（务必先读，否则下面会误读）

1. **D(n) 与 D(2m)=4m**
   - D(n) = n×n 网格上无三共线点的最大个数。中心猜想是 **D(n) ≥ 2n**。
   - "D(2m)=4m" 就是 D(n)=2n 在 n=2m（偶数）时的写法：在 (2m)×(2m) 网格放 4m=2·(2m) 点。
   - 两者是同一目标，没有矛盾。本文统一记为"能否对**所有偶数 n** 构造 2n 点解"。

2. **"center circumcenter"（本仓库的精确定义）**
   - 指：**网格中心 C 是解中某个三元组的外心**
   - 等价于：存在某个**距离环**（与 C 同距的点集）含 **≥3 个解点**。
   - **≠** "中心点 C 本身被占用为解点"。这两个概念互相正交：
     - C ∈ S 不会让 C 成为外心（C 到自身距离 0，与另两点不等距）。
     - C 是外心 ⟺ 有 3 个**其他**点共圆于 C，C 自己不必在解里。
   - 本仓库定理 2.1（C₄ ⇒ 非 missing-center）说的就是"共圆"，不是"占用 C"。

3. **missing-center（本仓库定义）**
   - 解中**没有任何**三元组以 C 为外心 ⇔ **每个距离环 ≤2 个解点**。
   - 与"是否占用 C"无关。

4. **对称类**（标准 D₄ 分类）
   - `rot4` = C₄（90° 旋转）；`rot2` = C₂（含 180° 半转 R₁₈₀）；
   - `dia2` / `rct4` = 含 R₁₈₀ 的四元群；
   - `iden` / `dia1` / `ort1` / `ort2` = **不含** R₁₈₀。

## 1. 方向 1（最高价值）：C₄ 构造存在性定理

### 精确问题
> 是否对**每一个偶数 n**，都存在一个 **C₄ 对称**的 2n 点解？
> 若成立 ⇒ 对所有偶数 m 有 D(2m) ≥ 4m，是 NTIL 中心猜想 D(n)≥2n 的一个重大**特例**结果。

### 我们已经有的（干净的起点）
- **C₄ 解的存在范围已知**：n = 6..56（Flammenkamp 连续），外加 n=58,60,62,64,66,68,70,72（rot4.few / Heule 2026）。**n=74 未知**（README line 315 明确：截至 2026-06-25 无已知 rot4 解；我们后来的 GPU 搜索在跑但未证实）。
- **图归约（定理 2.2 / 行-度等价）**：C₄ 的 2n 点解 ⇔ 在 m=n/2 个顶点上的 **2-正则图**（若干不相交圈），使得对应的 4m 个点无三共线。
  - 这一步把"几何存在性"成功转成了"图论 + 共线校验"问题——**用户说的"给了一半"正是这半**。

### "另一半"是什么（真正的硬骨头）
- 给定一个 2-正则图（圈分解），它诱导出的 4m 个点是否无三共线？这是**几何障碍**。
- 我们的 n=74 多次尝试（CaDiCaL / Z3 / 2-matching / 置换）**全部失败**——这强烈暗示障碍是真实的，但**不等于"无解"**（只是试过的图都碰到了共线）。

### 🔑 关键杠杆：循环普适性猜想（README §813）
> 对 C₄ 轨道选择问题，**full m-cycle** 与 **(1,m−1) cycle** 两种模式对所有 m ≥ 10 都有效（已验算 m=10..28，m=6,9 为例外）。这两种模式是否对**每个 m ≥ 10** 都生成合法 C₄ 解？

- 若此猜想**成立** ⇒ 我们立刻有了一个**显式构造**（取 full m-cycle 或 (1,m−1) 作为 2-正则图）⇒ **直接解决方向 1 对所有 m ≥ 10 的情形**。
- 这是方向 1 里**最具体、最可攻**的一环，比"一般存在性证明"现实得多。

### 价值 / 风险
- **价值最高**：若证成，是 D(n)≥2n 的重大部分结果（且是构造性的）。
- **风险**：可能对某些 m 失败（猜想已承认 m=6,9 例外，也许还有更多）。若假，找出障碍本身也是负结果（说明 C₄ 对称并非 extremal 解的普适结构）——同样有意义。

### 建议的第一步
1. 把方向 1 **等价重述**为"证明循环普适性猜想"。
2. 对 **full m-cycle** 模式尝试代数证明：
   - 中心方向互异（README line 53 的 distinct-direction law：每条过 C 的线至多含一个轨道的一对对顶点）；
   - 非中心线上无三共线（这是难点，需要找到不变量/模运算论证）。
3. 或先**补齐 n=58..72 之间的 C₄ 存在性**（已部分已知，确认完整，排除"小 n 缺口"干扰）。

## 2. 方向 2：missing-center 与对称性的关系

### 用户猜想
> 充分大的 extremal（2n）解都有 center circumcenter（即都是**非 missing-center**）。

### 我们已有什么（现成的干净定理）
- **定理 2.1 + 推论**：C₄ 对称 ⇒ 非 missing-center。
  ⇒ **每一个 missing-center 解都必然没有 C₄ 对称**。
  —— 这条直接就是方向 2 的现成定理，无需重新发明。

### ⚠️ 关键澄清：数据**不支持**用户的强猜想
- 数据：偶数 missing-center 存在于 **n=12..30**（52→11→103→345→…→534）；奇数一直延伸到 **n=53**。
- README line 318 说"n≥32 目录归零"——但**这是目录完整性假象**：大 n 只收录了 `rot4` 和 `dia2`，而 `rot4` 自动非 missing-center（定理 2.1）。不能据此说"大 n 都非 missing-center"。
- 结论：**用户那个强猜想目前既无证明、也无数据支撑**，应先收窄为可证的子命题。

### 更可攻、也更有价值的子问题（推荐优先级）
1. **对称分类定理（最推荐先做）**
   > missing-center 解可以具有除 C₄(`rot4`) 之外的**任何**对称类。
   - 证明：(⟸) 定理 2.1 排除 `rot4`；(⟹) 在每个非 C₄ 类（`iden`/`dia1`/`ort1`/`ort2`/`rot2`/`dia2`/`rct4`）里给出**至少一个** missing-center 实例（小 n 有限验证即可）。
   - 干净、低风险、可直接成文，且正是方向 2 的自然深化。

2. **偶数阈值 "为什么 n=12"**（README 已标 open，line 416）
   - 证明 n≤10 无 missing-center 目前仍是 heuristic（内环回避机制），**不是定理**。把它证出来，能把"n=12 阈值"变成严格结果。

3. **存在性本质问题**
   - missing-center 的 2n 解是否对任意大 n 都存在？是构造还是定理？
   - 现状：大 n 目录有缺口（`iden` 只到 n=20，`rot2` 在 n=31 后 UNSAT），**目前无法回答**，需先补数据或换思路。

### 价值 / 风险
- 对称分类定理：高价值 / 低风险（首选）。
- 强猜想本身：数据不支持，需先收窄，不宜直接强攻。

## 3. 方向 3：距离环超图（distance-ring hypergraph）理论

### 精确建模（用户直觉是对的，且可以做到精确）
- 顶点 **V** = 全部网格点。
- 超边 **E_ring** = 落在同一个以 C 为心的圆（同距环）上的 **3 点子集**。
- 则：**missing-center ⇔ S 是 H_ring 的独立集**。
  - 精确性：missing-center = 没有环含 ≥3 点 = 没有超边 ⊆ S。✅ 完全等价，无浮点。
- 完整 NTIL 解 = 在 **H = H_collinear ∪ H_ring** 中的独立集：
  - H_collinear：共线三元组超图（NTIL 标准表述，我们 `danger_hypergraph` 已建）。
  - H_ring：同外心三元组超图（**新增**）。
- "非 missing-center 的 NTIL 解" = H_collinear-独立，但**不是** H_ring-独立（含一个环超边）。

### 新颖性评估（诚实）
- **H_collinear（共线超图）是标准工具，非新。**
- **H_ring + "missing-center = 独立集" 刻画是新颖且干净的**：它把 missing-center 与非 missing-center 统一为超图独立集的二分，并把我们已有的 `danger_hypergraph`、orbit-ring 不变性、C₄ 定理串成同一语言。
- 一个漂亮推论：**C₄ 定理 = "C₄ 解永远不是 H_ring 的独立集"**。因为 orbit-ring 不变性使每个被使用的环人口 ≡ 0 (mod 4) ≥ 4，自动含超边。这正是方向 2 定理的超图语言重述。

### 价值 / 风险
- 作为**统一框架**价值高、风险低，适合作为论文的理论节 / 总纲。
- 可做的具体工作：
  1. 形式定义 H_ring，写出"missing-center = 独立集"引理（平凡但值得正式陈述）。
  2. 计算小 n 的独立数 / 染色数，观察增长。
  3. 把 orbit-ring 不变性、C₄ 定理、方向 2 的对称分类全部翻译成超图语言。

- **单解 cut-and-project 失败**：P2 半空间拟合准确率≈随机基线(79%–88%)，单个解不是低维格点干净截面。
- **投影角度新想法（highdim_angles.py）**：同n全部解 SVD 落于共享 k90 维子空间（形式上是"公共高维源在不同角度的投影"，SVD 保证，本身不深）。但 corr(PC1,距离环数)≈0（n=14/16/18: −0.03/+0.05/−0.11）；环数 silhouette≈0（线性/非线性均≈0）；随机投影角度未超过 SVD 最优。→ 解集在共享子空间是**均匀团块，无隐藏分层**。
- **公平性补充（highdim_nonlinear.py）**：第二标签"点对斜率多样性"(142–192) + 非线性 t-SNE 嵌入：corr(PC1,斜率)≈+0.056；线性/非线性 silhouette 环数/斜率均≈0。→ 线性与非线性视图下都无几何参数分层。
- **判定**："高维有序投影"假说**形式成立但强度弱**（共享子空间存在但只占38–62%D、仅薄~25%；团块无隐藏几何分层）。约束了该假说：若真有隐藏源，非由环数/斜率多样性在低维/非线性投影下组织。负担在"未测的代数/组合不变量或非线性度量流形"方。
