## §1 定义与可证引理

**记号**。S ⊆ [n]×[n]，|S|=2n，无三点共线。对空格 e，其**覆盖线集** L(e)={过 S 中某两点且经过 e 的直线}，覆盖度 d(e)=|L(e)|。因 S 合法，每条 S-点对线恰含 2 个 S-点，故 d(e) = 与 e 共线的 S-点对个数。

**定理 T1（最大性紧度）**。若 D(n)=2n，则 S 极大 ⇔ 其禁点集 F(S)（所有与某 S-点对共线的空格）恰为全部 n²−2n 个空格，即 \|F(S)\|=n²−2n。⇒ S 是共线超图极大独立集，且每个空格至少被一条 S-点对线穿过（d(e)≥1）。

**引理 T2（k-翻转刻画）**。设 A⊆S，|A|=k，B 为 k 个空格，T=S\A，S'=T∪B。则
> S' 合法  ⇔  对每个 b∈B，L(b) 中至多 1 条直线不含 A 的任意点。

*证明*。（⇒）若 S' 合法，b 与 T 中任两点不共线。若 L(b) 有两条线均不含 A 点，则它们上的 S-点对仍完整留于 T，b 与之共线，矛盾。（⇐）每 b 至多 1 条覆盖线避开 A ⇒ b 与 T 中至多 1 个 S-点共线（单点不构成共线）；移除 A 不引入新共线。故 S' 合法。□

**定理 Th-1（单线空格 ⇔ 单翻转边）** [E 的等价形式，取 k=1]。S 有单翻转邻居 ⇔ 存在空格 e 恰落在**唯一一条** S-点对线上（即 d(e)=1）。
> **⇒ 猜想 E（G₁ 无边）⇔ 对所有最大 2n 解 S、所有空格 e，d(e)≥2。**

**证明**。（⇒）若 e 合法翻转候选，由 T2 仅需其唯一覆盖线避开被移点 p，故该线必为 {p,q}，且 e 无其他覆盖线。（⇐）若 e 仅被一条线 L={p,q} 覆盖，则在 S\{p} 中唯一过 e 的线仅含 q，无其他 ≥2 点线过 e，故 (S\{p})∪{e} 合法。□

## §2 ★定理 Th-2（十字钉死，2026-07-10 已证）—— 取代原猜想 E

**定理 Th-2（十字钉死）**。任意最大 2n 解 S 的每个空格 e 都满足 d(e) ≥ 2（被至少两条不同的「恰含 2 个 S 点」的直线穿过）。

**证明**（三步）：
1. **每行、每列恰 2 个 S 点。** 无三点共线 ⇒ 任一行/列 ≤2 个 S 点；2n 点 / n 行（列）⇒ 恰 2 个。

## §8 (π,σ) 双函数分解框架与 L18-L20

每个 NTIL 解 S 唯一对应一对函数 π,σ：[n]→[n]，其中 π(i) < σ(i) 是第 i 行的两列坐标。
多重集 π∪σ 含每列恰 2 次（2-因子双射的列投影）。

**定理 10（列配对 2-正则性）**。列配对图 G=(V=[n], E={(π(i),σ(i))}) 中每列度数 ≤2。
证明：每列在 π∪σ 中出现 2 次 ⇒ 至多 2 条相异边。□

此框架下，NTIL 问题转化为：寻找函数对 (π,σ) 满足
1. π(i) < σ(i) — 规范顺序
2. π∪σ 每列恰 2 次 — 列约束
3. S = {(i,π(i))} ∪ {(i,σ(i))} 无三点共线 — 几何约束

条件 3 自动保证 π 和 σ 各自无三点共线（⊂ S 的子集性质）。

**新发现的实证律**：
- **L18（列配对多样性）**：avg_deg→2 说明大 n 解更倾向于多样化配对，小 n 更多"重复配对"。
- **L19（sigma|pi 刚性上界）**：当 pi(i) 接近 n-1 时 sigma(i) 几乎强制为 n-1——大列只与更大列配对。
- **L20（远距配对偏好）**：列配对倾向于互补对（如 (3,5)、(0,15)），相邻对相对罕见。

### §9.1 ★ 定理 Th-10：列非排列定理

**定理**：对 n ≥ 2，NTIL 解的 (π,σ) 均非 {0,...,n-1} 上的排列。

**证明核心**：
1. 若 π 是排列 ⇒ σ 也必是排列（每列总出现次数为 2）
2. 定义 τ = σ ∘ π^{-1} ⇒ 对所有 k，k < τ(k)（因 π(i) < σ(i)）
3. 对 k = n-1：τ(n-1) > n-1 ⇒ 矛盾（τ 的值域为 {0,...,n-1}）

**更强结论**：
- **列 0 恒左独占**：m_σ(0) = 0 ⇒ m_π(0) = 2（因 σ(i) = 0 要求 π(i) < 0 不可能）
- **列 n-1 恒右独占**：m_π(n-1) = 0 ⇒ m_σ(n-1) = 2（因 π(i) = n-1 要求 σ(i) > n-1 不可能）

**列划分**：记 k = |L| = |R|。列集划分为：
- L（左独占）：m_π=2, m_σ=0，恒含列 0
- B（平衡）：m_π = m_σ = 1
- R（右独占）：m_π=0, m_σ=2，恒含列 n-1

其中 k ≥ 1。

**验证**：205,869 个解、零反例。完整证明见 `theory_ps_nonpermutation.md`。

### §9.2 ★ 定理 Th-11：C₄ 互补配对定理

**定理**：rot4 解中 c ∈ L ⇔ n-1-c ∈ R。

**证明核心**：C₄ 对称性 σ̄(x,y) = (n-1-y, n-1-x) 诱导变换恒等式：
$$\pi(n-1-i) = n-1-\sigma(i), \quad \sigma(n-1-i) = n-1-\pi(i)$$
若 m_π(c) = 2（c ∈ L），则存在两行 i,j 使 π(i)=π(j)=c ⇒ σ(n-1-i)=σ(n-1-j)=n-1-c ⇒ m_σ(n-1-c) = 2 ⇒ n-1-c ∈ R。

**验证**：21,701 个 rot4 解、零反例。

### §9.3 解释"简单排列构造法"的必然失败

此前尝试的闭式构造 S = {(σ(i), i), (τ(i), i)} 要求 σ,τ 均为排列。Th-10 宣告：**任何这种构造在 n ≥ 2 时必然存在三点共线**。这完全解释了 `test_simple_construction.py` 随机 50,000 次零成功的负结果——不是搜索不够充分，而是数学上不可能。
