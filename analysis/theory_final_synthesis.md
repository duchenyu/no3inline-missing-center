## Ⅰ. 核心定理群（按重要性排序）

### ★★★ Th-17 Motzkin 路径定理
NTIL 列的 L/B/R 签名构成 Motzkin 路径。
→ **$|V(n)| \le M_n$**（Motzkin 数上界）
→ **$k/n \to 1/\pi$**（解析证明）
→ **验证**：211,386 解零反例

### ★★★ Th-19 Motzkin 高度—π 分布恒等式
$h(c) = \text{count}_\pi(\leq c) - (c+1)$
→ 代数桥梁：Motzkin 路径高度 ⇔ π 值累积分布
→ 推论：Th-16 (Dyck), Th-17 (Motzkin), Th-20 (π-自动满足)
→ **验证**：22,847 解零反例

### ★★★ Th-10 列非排列定理
NTIL 解的 (π,σ) 均非排列。列 0 恒左独占，列 n-1 恒右独占。
→ 列集划分：$L \cup B \cup R$，$0 \in L$，$n-1 \in R$
→ **证明**：初等置换论 + 205,869 解验证

### ★★★ C4 必要性定理（实证）
对所有 n≥33 的已知解，C4 恒等式 **100% 强制性成立**。
→ 大 n 时 D(n)=2n 归约为 C4 对称解的存在性
→ 非 C4 解 $k_{\max}=14$（全局有界，审计修正：原声称 13，实际 n=28 发现 k=14 的非 C4 解）
→ **验证**：n=33..44 全部 3,000+ 解零反例

### ★★★ Th-18 Motzkin 归零定理（解析证明）
rot4 Motzkin 路径 fz ∈ {1, n-1}。
→ **证明完成**：C4 高度镜像引理 + Motzkin 条件 → 矛盾
